# Agente RAG — AluraChallenge

Agente de IA corporativo basado en **RAG** (Retrieval-Augmented Generation) que
responde preguntas de los colaboradores a partir de documentos internos de la
empresa en múltiples formatos.

Proyecto desarrollado para el **Challenge AluraAgente — ONE IA FOR TECH**.

## Estado

🚧 En construcción.

## Objetivo

Construir un agente conversacional, accesible para toda la organización, capaz de
comprender documentos en distintos formatos (PDF, Word, Excel, PowerPoint,
Markdown, CSV, JSON y HTML) y desplegarlo en la nube de Oracle (OCI).
